This project is a recreation of the official SDU University website as it appeared in 2023. It was created primarily to demonstrate and practice my HTML and CSS skills at that stage of my learning journey.

The goal was to replicate the website's design and layout as accurately as possible while applying the knowledge and techniques I had learned. Although the project is not fully complete and still contains areas for improvement, it serves as a reflection of my progress and understanding of front-end development at that time.
